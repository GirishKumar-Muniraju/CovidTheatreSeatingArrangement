How to use Seating Allocation Engine

Scenario(1):

By Default, we have set a initial 2x2 block with 50% allocation.

Scenario(2):

On Changing theatre & govt allocation occupancy

-- On updating Total seats in theatre. Total seats avaible is dynamically calculated
-- On updating no.of blocks. New set of rows & cols input field will be enabled for user with default value.
-- When block level rows & cols are enabled, user has ability t update it or retain default mXn matrix's

Scenario(3):

On changing block level rows & cols

-- When block level rows & cols are enabled, user has ability t update it or retain default mXn matrix's

In any case, Click on Show - Seating Arrangement button to see seating arrangement.
